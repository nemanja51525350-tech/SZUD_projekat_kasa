# SZUD_projekat_kasa
Fakultetski Projekat Koji dokaziuje da sam sposoban za osnovno rukovanje gitom

# Uputstvo :
1. Pokrenuti program
2. Uneti broj artikala
3. Za svaki artikal uneti:

a. naziv

b. količinu

c. cenu

5. Izabrati način plaćanja:

y → plaćanje kešom

n → plaćanje karticom

6. Ako je kartica:

d → plaćanje debitnom

k → plaćanje kreditnom

7. Ako je keš:
uneti iznos koji je kupac platio
8. program automatski izračunava kusur
9. Račun se ispisuje na ekranu i snima u fajl
# Fajlovi

main.c – izvorni kod programa

Fisk.txt – tekstualni fajl sa sačuvanim fiskalnim računima
(automatski se kreira ako ne postoji)
# Kompatibilnost 
Program je kompatibilan sa:

GCC

Dev-C++

Code::Blocks
# Novo:
Uzeto je u vidu da li kupac koristi debitnu ili kreditnu karticu, i stanje njegovoga racuna.
